# HealthFirst Patient Portal 🏥

A modern, responsive patient portal built with React, Vite, and Lightswind UI components. This application provides patients with secure access to their health records, appointments, and medical information.

## ✨ Features

### 🔐 Authentication System
- **Secure Login** with demo credentials
- **Multi-step Registration** with form validation
- **Password visibility toggle** for better UX
- **Remember me** functionality
- **Responsive design** for all devices

### 📊 Patient Dashboard
- **Health metrics overview** with animated counters
- **Upcoming appointments** with detailed information
- **Recent activity** notifications
- **Quick actions** for common tasks
- **Health summary** with vital signs
- **Search functionality** for medical records

### 🩺 Medical Records
- **Lab results** with trend indicators
- **Vital signs** monitoring
- **Medication management** 
- **Health status indicators**
- **Export and sharing** capabilities
- **Advanced filtering** and search

### 🎨 Modern Design System
- **Lightswind UI components** for consistency
- **Glass morphism effects** and modern aesthetics
- **Animated backgrounds** with interactive elements
- **Gradient borders** and smooth transitions
- **Responsive layouts** for mobile-first design
- **Accessibility features** with proper focus states

## 📦 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd patient-portal
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

## 🧪 Testing

Run the test suite:
```bash
npm test
```

Run tests in watch mode:
```bash
npm run test:watch
```
